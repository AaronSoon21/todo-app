# 📝 To-Do App

A minimalist and interactive To-Do App built with JavaScript.

## 🚀 Features
- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Filter (All / Active / Completed)
- Clear completed tasks
- Dark mode toggle 🌙
- Data persistence using localStorage

## 🛠️ Built With
- HTML
- CSS
- JavaScript (Vanilla)

## 💡 What I Learned
- DOM manipulation
- Event handling
- Array methods (filter, find)
- State management
- LocalStorage usage
